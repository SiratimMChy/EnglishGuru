# 🌐 English Guru

![Premium Design](https://img.shields.io/badge/Design-Premium-skyBlue)
![Platform](https://img.shields.io/badge/Platform-Web-emerald)
![Status](https://img.shields.io/badge/Status-Active-blue)

**English Guru** is a high-fidelity, interactive English learning platform designed to help students master vocabulary, grammar, and pronunciation through a premium user experience. Built with a unified "Sky Blue" design system, it offers dynamic daily updates and curated educational content.

---

## ✨ Key Features

### 📅 Daily Educational Updates
-   **Common Mistakes:** A daily-rotating grid showing common English errors and their professional corrections.
-   **Grammar Mythbusters:** Interactive 3D flip-cards that debunk common language myths with logic and examples.
-   **Dynamic Card Tags:** Daily labels (e.g., "Monday's Highlights") that refresh every 24 hours to keep the content feeling alive.

### 📚 Interactive Vocabulary Learning
-   **Lesson-Based Learning:** Access multiple lessons with curated word lists.
-   **Advanced Injection Logic:** Custom-enriched lessons (like Lesson 7) featuring professional-grade terminology (Eloquent, Pragmatic, etc.).
-   **Word Fact Modals:** Deep-dive into any word with its meaning, pronunciation, real-world examples, and synonyms.
-   **Audio Pronunciation:** Built-in text-to-speech engine to hear the correct pronunciation of every word.

### 🎥 Multimedia & Insights
-   **Master English with Video:** A cinema-style video grid featuring curated tutorials from world-class educators (BBC, Rachel's English, etc.).
-   **Immersive Player:** Watch lessons directly on the site via a high-performance YouTube modal.
-   **Latest Insights (Blog):** A professional 6-card grid featuring expert strategy, business etiquette, and future trends (2024-2026).

---

## 🛠️ Technology Stack

-   **Structure:** Semantic HTML5
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
-   **Logic:** Modern Vanilla JavaScript (ES6+)
-   **Iconography:** [Font Awesome 6+](https://fontawesome.com/)
-   **Typography:** Google Fonts (Inter, Bengali-specific fonts)
-   **API Integration:** Programming Hero OpenAPI

---

## 🎨 Design Philosophy

-   **Visual Symmetry:** Standardized container widths and vertical margins for a perfectly balanced layout.
-   **Unified Color Palette:** A strict "Sky Blue" theme (`sky-500` primary) with emerald and slate accents.
-   **Micro-interactions:** Smooth hover effects, 3D card flips, and modal transitions to ensure a premium feel.

## 🚀 Getting Started

